# Working Specifications

> Thư mục này chứa các tài liệu đang làm việc (working documents).

## Mục đích

- Lưu trữ các phiên bản nháp, kế hoạch, và phân tích chi tiết.
- Các file trong đây có thể được di chuyển lên `docs/specs/` khi đã ổn định.
- Nên review định kỳ và dọn dẹp file cũ.

## Lưu ý
- File trong thư mục này **không** nên được coi là tài liệu chính thức.
- Khi hoàn thiện, hãy di chuyển file quan trọng lên thư mục cha (`docs/specs/`).
